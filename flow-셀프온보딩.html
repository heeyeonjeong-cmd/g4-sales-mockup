핵심 포인트: 뒷단에서 일어나는 일
사장님은 "가입 → 계약 → 포스 설치 → 테스트 주문"만 경험하지만, 뒤에서는:
1) 시스템이 배달 가능 여부 · 사업자 진위 · 기가입 여부 · VAN/더존 코드 발급 · 리드 스코어링 · 상점 계정 생성까지 전부 자동으로 처리하고,
2) 영업사원이 리드를 확인 → 포스 설치 → 초기 케어까지 마무리 터치를 담당하며,
3) 관리자는 온보딩 KPI를 통해 성과를 모니터링합니다.
→ 별도 승인 단계 없이, 전자서명 완료 즉시 시스템이 자동으로 모든 셋업을 처리하는 구조.
Step-by-Step
7단계 상세 플로우
사장님
Step 1. 랜딩 & 요금 시뮬레이션
사장님이 검색/광고/지인 추천으로 부릉 사장님 사이트에 도착합니다.
주소만 입력하면 배달 가능 여부가 즉시 확인되고, 예상 요금과 경쟁사 대비 절감액이 시각화됩니다.
"지금 가입하면 3개월 할인 요금제 4,300원 → 3,800원" CTA 노출.
⚡ 시스템: 주소 → 권역 매핑 → 라이더 수급 체크 → 배달 불가 시 "상담 신청"으로 전환
소요: ~2분
사장님
Step 2. 회원가입 & 사업자 인증
사업자등록번호, 대표자명, 연락처, 이메일 입력. SMS 인증.
사업자 진위를 국세청 API로 실시간 자동 확인. 휴업/폐업은 즉시 안내.
⚡ 시스템: 사업자 진위 API 호출 → 기가입/해지 이력 조회 → 중복 가입 방지
소요: ~3분
사장님
Step 3. 계약 정보 입력 & 전자 서명
요금제 선택 (시뮬레이션 결과 자동 반영), 배달 희망 권역, 영업시간 입력.
약관 동의 + 전자서명. 계약서 PDF가 자동 생성되어 이메일로 즉시 발송됩니다.
⚡ 시스템: 웹 계약서 확정 → VAN 코드 자동 발급 → 더존 코드 자동 발급 → 리드 스코어링 (예상 물량 산출)
소요: ~5분
시스템 자동
Step 4. 자동 분류 · 배정 · 상점 계정 생성
전자서명 완료 즉시 시스템이 자동으로 처리:
• VAN 코드 자동 발급 (카드 결제 수신용) + 더존 코드 자동 발급 (정산/세무 연동용)
• 리드 스코어링 → 물량 등급 판정
• 소물량 (~100건/월): 일반 영업사원 배정
• 앵커 (500건+/월): 시니어 영업사원 우선 배정
• 메가 (1,000건+/월): Key Account 매니저 직접 배정
• 상점 계정 자동 생성 + 권역 세팅 + 포스 연동 준비
영업사원에게 즉시 알림 (앱 푸시 + 세일즈 툴 알림). 사장님에게 "계약 완료! 포스 설치 일정을 잡아주세요" 안내 발송.
⚡ 시스템: VAN/더존 코드 발급 → 상점 계정 생성 → 물량 등급 판정 → 영업사원 매칭 → 알림 발송 (전부 자동, 승인 없음)
소요: 즉시 (자동)
영업사원
Step 5. 포스 설치 & 연동
사장님과 일정 조율 후 현장 방문:
• 포스 설치 (헥토 연동)
• 사장님사이트 사용법 안내
• 주문 수신 테스트
포스리스 옵션이 가능한 경우 원격 세팅으로 대체 가능 (향후)
소요: 30분~1시간
사장님 + 영업사원
Step 6. 테스트 주문 & 활성화
포스에서 테스트 주문 발생 → 라이더 배정 → 정상 배달 확인.
성공 시 상점 상태를 "활성(Active)"으로 전환.
⚡ 시스템: 테스트 주문 성공 → 상점 상태 Active 전환 → 활성화 축하 메시지 발송
소요: ~15분
시스템 + 영업사원
Step 7. 초기 30일 모니터링 (이탈 방지)
활성화 후 30일간 이탈 모니터링:
• 3일 연속 주문 0건 → 영업사원에게 이탈 경보
• 7일 무주문 → 사장님에게 "어떤 어려움이 있나요?" 자동 발송
• 앵커 상점 → 전담 매니저가 주 1회 체크인 콜
⚡ 시스템: 주문 패턴 모니터링 → 이탈 예측 스코어 산출 → 알림 자동 발송
