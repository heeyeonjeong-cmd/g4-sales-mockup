Step-by-Step + Mockup Mapping
9단계 상세 플로우
각 단계에서 세일즈 웹의 어떤 화면을 사용하는지 표시했습니다.
영업사원
Step 1. 리드 발굴 & 현장 방문
타겟 상점 리스트를 세일즈 웹 지도 뷰에서 확인하고 방문 동선을 계획합니다.
플랫폼 다중 입점 상점, 가게배달 미사용 매장 등을 우선 타겟팅.
앵커 상점(500건+/월)이면 리드 스코어가 높게 표시 → 우선 방문.
소요: 이동 포함 ~30분/상점
영업사원
리드 상세
Step 2. 리드 등록 & 정보 확인
세일즈 웹 리드 상세 화면에서:
• 상점 기본 정보 입력 (상점명, 대표자, 연락처, 주소, 사업자번호)
• 시스템이 사업자 진위 + 기가입/해지 여부 자동 확인
• 방문 이력, 영업 메모 기록
• 월 예상 주문수, 관심도 표기
⚡ 시스템: 사업자번호 → 국세청 API 진위 확인 → 기가입 체크 → 배달 가능 권역 확인
소요: ~5분
영업사원 + 상점주
요금 시뮬레이션
Step 3. 요금 시뮬레이션 & 피칭
세일즈 웹 요금 시뮬레이션 화면을 상점주에게 보여주며:
• 요금제 선택 (기본 4,300원 / 할인 3,800원 / 특별 3,500원)
• 할증 조건 선택 (우천, 심야, 장거리, 성수기)
• 희망 배송 권역 + 라이더 수급 상태 확인
• 경쟁사 대비 절감액이 자동 계산되어 시각화
상점주: "월 26만원 절감" 같은 구체적 숫자 확인 → 계약 의향 상승
⚡ 시스템: 요금제 × 월 주문수 → 예상 비용 자동 계산 + 경쟁사 비교 테이블 생성
소요: ~10분 (설명 포함)
영업사원 + 상점주
계약서 생성
Step 4. 현장 계약서 생성 & 서명
"계약서 생성으로 이동" 버튼 클릭 → 계약서 생성 화면:
• 상점 정보 + 시뮬레이션 결과가 자동 반영 (수작업 0)
• 계약 기간 선택 (6/12/24개월)
• 특이사항 기재 (첫 달 무료, 포스 무상 등)
• 계약 요약 확인 → 상점주가 화면에서 터치 서명
→ 이전: 별도 계약서 준비 → 재방문 필요 → 2~3일 소요
→ 이후: 현장에서 즉시 완료 → 5분
⚡ 시스템: 입력 데이터 → 계약서 PDF 자동 생성 → 할인 요금 시 승인 필요 플래그
소요: ~5분
영업 관리자
승인 관리
Step 5. 관리자 승인
승인 관리 화면에서 관리자가 검토:
• 승인 대기 목록에서 계약 조건 확인 (요금제, 월 예상, 권역, 기간)
• 할인율 적정성 판단 → 승인 or 반려 (사유 필수)
• 앵커 상점 특별 요금 → 팀장 확인 권고 표시
• 기본 요금(4,300원)이면 자동 승인 → 관리자 개입 불필요

승인 후 자동 처리:
① 계약서 확정 + PDF 발급
② S-form 자동 생성/전송
③ 상점주에게 카카오톡/SMS 안내 발송 (계약 완료 + 사장님사이트 가입 링크 + 헥토 가이드)
⚡ 시스템: 승인 즉시 → S-form 생성 → 상점 계정 생성 → 안내 메시지 일괄 발송
소요: ~30분 (SLA 목표)
시스템 자동
계약 완료
Step 6. 계약 완료 & 자동 셋업
계약 완료 화면에서 확인 가능:
• 계약서 PDF 미리보기/다운로드/이메일 전송
• S-form 자동 생성 상태 (상점 코드, 요금제 코드, 권역 코드)
• 승인 이력 타임라인
• 상점주 안내 발송 현황 (카카오톡 계약 완료 알림, SMS 가입 링크, 헥토 가이드, 포스 안내)
⚡ 시스템: 상점 마스터 등록 → 권역/요금제 세팅 → 사장님사이트 계정 준비
소요: 즉시 (자동)
상점주
Step 7. 사장님사이트 가입 & 헥토 연동
상점주가 SMS 링크로 사장님사이트 회원가입:
• 이미 생성된 상점 계정에 매핑
• 헥토(결제) 가입 연동 진행
이 단계에서 연령대 높은 사장님은 어려움 겪을 수 있음 → 영업사원이 전화/재방문 지원
소요: ~15분 (사장님 직접)
영업사원
온보딩 추적
Step 8. 포스 설치 & 테스트 주문
온보딩 추적 화면에서 진행 상황 체크:
• 포스 설치 일정 조율 → 재방문 (현재 계약과 분리되어 있는 핵심 병목)
• 헥토 포스 설치 + 연동 확인 (30분~1시간)
• 사장님사이트 사용법 안내
• 테스트 주문 실행 → 정상 배달 확인

온보딩 체크리스트: ✅ 계약 완료 → ✅ S-form → ✅ 사장님사이트 가입 → ✅ 헥토 가입 → ✅ 포스 설치 → ✅ 테스트 주문
소요: 30분~1시간 (재방문 필요)
시스템 + 영업사원
온보딩 추적
Step 9. 활성화 & 30일 모니터링
테스트 주문 성공 → 상점 상태 "Active"로 전환.
이후 30일간:
• 시스템이 주문 패턴 자동 모니터링
• 3일 연속 주문 0건 → 영업사원에게 이탈 경보
• 앵커 상점 → 전담 체크인 콜 (주 1회)
• 온보딩 추적 화면에서 전체 상점의 온보딩 상태 / D+7 주문율 확인
